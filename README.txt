Технологии
Windows Forms - для создания пользовательского интерфейса
MS SQL Server - для хранения данных
ADO.NET - для взаимодействия с базой данных

Данное приложение разработано в рамках учебного проекта для автоматизации учета клиентов и заказов в малом бизнесе. Программа предназначена для упрощения процессов управления клиентской базой и заказами, обеспечивая удобный интерфейс и надежное хранение данных

Основные задачи проекта
Освоение клиент-серверной архитектуры
Работа с базой данных через ADO.NET
Создание удобного интерфейса для управления данными
Применение SQL-запросов и хранимых процедур на практике

Функционал
1. Управление клиентами
Добавление, редактирование и удаление клиентов
Хранение контактных данных (Имя, Email, Номер телефона, Адрес)
Автоматическое проставление даты создания клиента

2. Управление заказами
Создание заказов с привязкой к клиентам
Указание суммы, статуса и описания заказа
Доступные статусы заказов (Новый, В работе, Завершен, Отменен)

Структура базы данных
Приложение использует две основные таблицы

Клиенты
Содержит данные о клиентах (Имя, Email, Номер телефона, Адрес, Дата создания)
Заказы
Содержит информацию о заказах (Клиент, Сумма, Статус, Описание, Дата создания)
